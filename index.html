Este proyecto se trabajo el análisis de datos del catastro turístico que proporciona el Ministerio de Turísmo, donde en la Matería de Bases de Datos se hizo la limpieza, agregación de datos y inserción de los datos una base de datos en MySQL, para luego en la Materia de Programación Avanzada se realizo la conexión con la base de datos y se formulo diversas consultas útiles para los usuarios, una vez hechas las consultas se configuro para que estas se muestren en gráficos, que son mas entendibles para los usuarios finales.
