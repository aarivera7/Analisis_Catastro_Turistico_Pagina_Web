¿De qué trata?

En este proyecto se trabajo el análisis de datos del catastro turístico que proporciona el Ministerio de Turísmo, donde en la Matería de Bases de Datos se hizo la limpieza, agregación de datos y inserción de los datos una base de datos en MySQL, para luego en la Materia de Programación Avanzada se realizo la conexión con la base de datos y se formulo diversas consultas útiles para los usuarios, una vez hechas las consultas se configuro para que estas se muestren en gráficos, que son mas entendibles para los usuarios finales.

¿Cuáles fueron los resultados?

Los resultados obtenidos fueron diversas gráficas que sirven para ver el estado del sector turístico, en las diferentes áreas que existen, como por ejemplo que área turística ya se encuentra saturada, o cuál es la que menos competencia tiene, estos datos pueden ser muy utiles para las empresas que quieran expandirse o crear nuevos negocios, y también sirver para conocer en que lugares se debe establecer determinada actividad economica para tener mayor exito.
